#### Status
Under construction 
#### Commands
<pre>
new       &lt;DB NAME&gt;
          Create new a DB
delete    &lt;DB NAME&gt;
          Delete a DB
dbs       [DB NAME] .. [DB NAME]
          Show all DBs or some DBs
ext       list
          Show all Exts
ext       add &lt;EXT FILE (.json)&gt;
          Add a Ext
ext       delete &lt;EXT ID&gt;
          Delete a Ext
node      list
          Show all Nodes
ext       delete &lt;NODE ID&gt;
          Delete a Node
</pre>
